Título do projeto: Sistema de Controle de Ações

Status do projeto: em desenvolvimento

Tecnologias que serão aplicadas: Java, MySQL, GitHub, Git

Time de desenvolvedores: Rodolfo Gontijo

Objetivo do software: O objetivo do sistema é permitir que investidores registrem e acompanhem suas operações de compra e venda de ações de forma simples, organizada e intuitiva.
O sistema será desenvolvido para ambiente desktop e servirá como ferramenta de controle da carteira de investimentos do usuário.

Funcionalidades do sistema: 
    • Cadastrar ações da bolsa de valores;
    • Registrar operações de compra;
    • Registrar operações de venda;
    • Consultar as ações;
    • Consultas as Vendas;
    • Consultar as Compas.
